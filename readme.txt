Manager jquery slider by options drag/ drop. Support all most responsive jquery slider have options: bxSlider, flexSlider, nivoSlider, swiper slider, bootstrap and your custom slider,.... Support php, shortcodes

== Description ==

**If We Have a Option Slider, We Can use `Slider Options` Manager it by Drag and Drop. ;)**

= Features: =
* Drag and Reorder Slides
* Upload/Delete image
* Add Hyperlinks to Slides
* Set Link Target for Each Slide
* Shortcode Support
* Video ( Youtube or Vimeo Video,... ) as Slides
* Internalization Support
* Customization Your Option: animation, caption, responsive, control buton, ...
* Use minify resources

== Installation ==

1. Upload your plugin folder to the `/wp-content/plugins` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Look at your admin bar and enjoy using the new Menu item (SM Slider)
